# Agentforce Agent Assets, External Services, and Named Credentials
# This file contains metadata types for Agentforce-related assets
# Reference file for understanding what Agentforce metadata types are included in this project
#
# Note: EinsteinAgentConfig is not available via Metadata API
# Agent configurations may be stored in Custom Objects or Custom Metadata
#
# This project includes the following Agentforce assets:
# - Bot and BotVersion (agent configurations - Agent metadata type not yet supported in CLI)
# - GenAI Functions (actions that can be added to agents)
# - GenAI Plugins (topics/categories of actions)
# - GenAI Planner Bundles (agent planner configurations)
# - Permission Sets (user access to agents/bots)
# - Profiles (user access to agents/bots)
# - Flows (including flow actions)
# - External Services and Credentials
